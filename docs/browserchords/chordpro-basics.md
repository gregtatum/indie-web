# ChordPro basics

Browser Chords supports common ChordPro features for labeling titles, sections,
and chord annotations.

## Example

```
{title: 500 Miles}
{artist: The Proclaimers}

[C]When I [G]wake up
```

## Tips

- Keep metadata at the top for quick parsing.
- Use section headers like `{verse}` and `{chorus}` for structure.
